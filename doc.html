Load css

		$loader->load('resourcesloader')->loadcss(<directory, with defaults>,<url, default false>);

css directory (dir) - default folder "css", default async folder "async/css". You can specify your own directory e.g. "assets/css" or custom directories for css and asynchronous css e.g. 

		$dir = [
			'css' => 'assets/css',
			'async' => 'assets/css/async'
		];

		$loader->load('resourcesloader')->loadcss($dir);

url - if no url provided then the url from the module config will be use instead, this will define the url location of your assets;

either one of the set up exist in array or not found, default directories defined on the module config will be use instead.


Load Js

		$loader->load('resourcesloader')->loadjs(<directory, with defaults>,<url, default false>,<config, default []>);

js directory (dir) - default folder "js", default async folder "async/js". You can specify your own directory e.g. "assets/js" or custom directories for js and asynchronous js e.g. 

		$dir = [
			'js' => 'assets/js',
			'async' => 'assets/js/async'
		];
		$loader->load('resourcesloader')->loadjs($dir);

url - if no url provided then the url from the module config will be use instead, this will define the url location of your assets;

config array - this where you can define your custom asynchrounous javacript files and parallel loading e.g. 

		$config = [
			'https://mywebsite.com/assets/js/custom/main.js', // loaded async
			[
				'parent' => 'https://jquery.com/api/cdn/jquery.min.js', // parent script
				'src' => [						     // you can provide an array of dependencies or string dependency url
					'https://mywebsite.com/assets/js/custom/jquery1.js', // dependecy
					'https://mywebsite.com/assets/js/custom/jquery2.js', // dependecy
					'https://mywebsite.com/assets/js/custom/jquery3.js', // dependecy
				];
			]
		];
